1. Verifies python3 + git on PATH (distro-aware error messages for missing dependencies).
2. Creates `.venv` + installs 4 runtime dependencies.
3. Invokes `em-cli setup` which auto-detects greenfield-vs-map + initializes `forge.config.yml` + chains `em-cli doctor` + emits "em-cli setup COMPLETE".
